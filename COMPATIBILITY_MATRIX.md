# Compatibility matrix

Fecha: 2026-04-30

## Navegadores objetivo
- Chrome (Windows/macOS/Android): ultimas 2 versiones
- Edge (Windows): ultimas 2 versiones
- Firefox (Windows/macOS): ultimas 2 versiones
- Safari (macOS/iOS): ultimas 2 versiones

## Dispositivos
- Desktop 1366x768, 1920x1080
- Tablet 768x1024
- Mobile 360x640, 390x844

## Casos base
- Navegacion de rutas principales
- Mapa georreferenciado
- Formulario de contacto
- Login admin
- CRUD admin (evento, galeria, hospedaje)
